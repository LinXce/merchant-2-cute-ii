

# [STS2 mod] 商人娘化mod2

本项目的贴图基于AI生成并进行了部分修改。

# mod使用方法

1. 下载 [Releases] 的Merchant2CuteII压缩包 解压到杀戮尖塔2游戏目录下的mods文件夹。
2. 如果没有mods文件夹可以手动创建。

该版本只包含商人部分，不包含假商人。（储君娘化mod为Mesugaki_Regent）
