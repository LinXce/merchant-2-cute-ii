# [STS2 mod] 商人娘化mod2

本项目的贴图基于AI生成并进行了部分修改。

# mod使用方法

1. 下载 [Releases] 的Merchant2CuteII压缩包 解压到杀戮尖塔2游戏目录下的mods文件夹。
2. 如果没有mods文件夹可以手动创建。

### v1.3.0-260503

更新配音
命令行:
merchant voice default|zh|jp
切换原版、中文、日语配音

### v1.2.0-260501

更新hand-foot切换。
按( ` )键打开杀戮尖塔2的控制台，输入merchant hand或者merchant foot进行切换。
